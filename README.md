# JS Clock

구형 스마트폰이나 태블릿을 활용하여 심플하고 직관적인 **디지털 시계**로 사용할 수 있도록 개발된 PWA(Progressive Web App)입니다.

## 🎯 개발 배경 및 목적
시중에 Flip Clock 등 다양한 탁상시계 앱들이 존재하지만, 대부분 광고가 덕지덕지 붙어 있거나 정작 필요한 기능은 빠진 채 불필요한 기능들로 무거운 경우가 많습니다.

특히 밤에 탁상시계로 켜놓을 경우, 아주 미세한 불빛조차 숙면을 방해할 수 있습니다. 따라서 **화면 밝기를 손쉽게, 그리고 극한까지 줄일 수 있는 기능**을 최우선으로 두고, 정말 꼭 필요한 기능만 담아 가볍고 최적화된 시계를 만들게 되었습니다.

## ✨ 주요 기능
- **직관적인 밝기 조절 (핵심 기능)**
  - 화면(상/하단 10% 제외) 어디서든 손가락으로 **위/아래로 문지르면(드래그)** 즉각적으로 시계의 밝기가 조절됩니다. 
  - 취침 전후에 눈부심이 없도록, 물리 버튼이나 상태바를 조작할 필요 없이 제스처만으로 화면을 가장 어둡게 설정할 수 있습니다.
- **오프라인 동작 지원**
  - PWA 기술이 적용되어 기기에 한 번 설치(홈 화면에 추가)하면, 와이파이나 **인터넷이 끊긴 오프라인 환경**에서도 완벽하게 작동합니다.
- **화면 꺼짐 방지 (Wake Lock)**
  - 시계가 켜져 있는 동안에는 기기의 화면이 자동으로 꺼지지 않고 계속 켜진 상태로 유지됩니다. (구형 기기를 위한 더미 비디오 대체 로직 포함)
- **앱 설치 (PWA 지원)**
  - 구글 플레이스토어나 앱스토어를 거치지 않고, 브라우저 메뉴에서 **'홈 화면에 추가'**를 선택해 네이티브 앱처럼 설치 및 전체화면으로 쾌적하게 사용할 수 있습니다.

## ⚙️ 설정 화면
우측 하단의 톱니바퀴 아이콘을 눌러 개인 취향에 맞게 디스플레이를 설정할 수 있습니다.

- **Show Date**: 날짜(연, 월, 일, 요일) 표시 여부를 켜거나 끕니다.
- **Show Seconds**: 초 단위 표시 여부를 켜거나 끕니다.
- **24-Hour Format**: 12시간제(AM/PM)와 24시간제 표시 형식을 전환합니다.
- **Font Size**: 하단 슬라이더를 통해 시계의 텍스트 크기를 자유롭게 조절합니다.
- **수동 업데이트**: 설정창 상단의 버전 번호(예: `v0.2.15`)를 터치하면 서버에서 최신 버전을 확인(`Checking...`)하고, 있다면 즉시 새로고침하여 앱을 업데이트합니다.

## 🚀 접속 및 설치 방법
1. 태블릿 또는 스마트폰의 브라우저(크롬, 삼성 인터넷, 사파리 등)에서 다음 링크로 접속합니다.
   - 👉 **[JS Clock 바로가기](https://socier.github.io/jsclock/)**
2. 브라우저 메뉴에서 **[홈 화면에 추가]** 또는 **[앱 설치]**를 선택합니다.
3. 바탕화면에 생성된 **JS Clock** 아이콘을 눌러 쾌적한 전체화면으로 실행합니다.
